To get started, you have to install `mason_cli` first

```shell
dart pub global activate mason_cli
```

or using `homebrew`

```shell
brew tap felangel/mason
brew install mason
```

### Instalation
You can install this brick globally, so you don't need to initialize mason in every project afterwards

```shell
mason add indcana_core -g --git-url https://github.com/homindolentrahar/indcana --git-path bricks/indcana_core
```

### Usage

Run this command to create new Flutter project, complete with boilerplate

```shell
mason make indcana_core
```
